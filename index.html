Mokymosi Kursas

JAVA

Java – bendrosios paskirties, objektiškai orientuota programavimo kalba, kurią 1991 m. sukurė Džeimsas Goslingas ir kiti „Sun Microsystems“ inžinieriai, o 2010 m. ją įsigijo Oracle korporacija.
Apie Javą oficialiai paskelbta 1995 metų gegužės 23 d., o išleista tų pačių metų lapkritį. Java (pradžioje vadinta Oak) kalbos pirminis tikslas buvo pakeisti C++ kalbą.
Javos aplikacijos yra kompiliuojamos į baitkodą, kuris gali būti paleidžiamas bet kokioje Javos virtualiojoje mašinoje (JVM), nepriklausomai nuo kompiuterio architektūros.
2015 m. Java yra viena iš populiariausių naudojamų programavimo kalbų, ypač kliento-serverio „web aplikacijose“. Ja taip pat rašomos programos Android išmaniesiems telefonams.

C#

C# (tariama sy šarp) – objektiškai orientuota programavimo kalba, sukurta Microsoft kompanijoje kaip dalis .NET iniciatyvos. Kalba paremta C++ bei Java kalbomis. Kalba kurta balansuojant tarp galingumo (C++ įtaka) bei greito programavimo (Java, Visual Basic įtaka).
Kalbos kūrėjai – Anders Hejlsberg ir Scott Wiltamuth (abu iš Microsoft).
Microsoft pateikė kalbos specifikaciją standartų organizacijai Ecma International (buvusiai ECMA) 2000 metais. 2000 m. liepos 11 dieną Microsoft išplatino pranešimą spaudai, skelbiantį apie naujas apibendrintas NET programavimo kalbas.
Nuo to laiko, kai Microsoft 2002 metais išleido pirmąją .NET Framework versiją, buvo atlikti du nedideli pataisymai ir vienas didelis, pateikiami su Windows atnaujinimais (Service Packs). Su pirmąja .NET versija išleista ir pirmoji C# kompiliatoriaus versija. Vėlesnės C# versijose šiek tiek praplėsta sintaksė.

HTML

HTML – tai paprasta ir lengvai išmokstama kalba tinklapiams kurti. Naujausia šios kalbos versija pastaruoju metu tampa viena populiariausių kalbų multimedijai perduoti. Su keliolika šios kalbos žymų galima sukurti visą tinklapį. Dauguma šiuolaikinių HTML rašyklių yra pakankamai funkcionalios, todėl nebereikia rašyti ilgų puslapių kodų.

CSS

CSS programavimo kalba išplečia HTML kalbos galimybes. Su šia kalba galima atvaizduoti teksto šriftą, tinklapio išdėstymą, spalvas, atstumą tarp elementų. CSS kalba leidžia tą patį stiliaus šabloną naudoti visuose tinklapio puslapiuose.

JavaScrip

Šiuolaikinei internetinei svetainei sukurti vien HTML kalbos neužteks. Būtent „Javascript“ kalba padės sukurti patogią naudoti, dinamišką svetainę. Su šia kalba galima kurti išskleidžiamuosius meniu, patvirtinimo lenteles, animaciją ir pan. – viską, kas suteiktų pridėtinės vertės jūsų svetainei.

PHP

PHP – plačiai paplitusi dinaminė interpretuojama programavimo kalba (en: Hypertext Preprocessor), sukurta 1995 m. ir specialiai pritaikyta interneto svetainių kūrimui.
PHP sintaksė panaši į daugelį struktūrinių kalbų, ypač į C bei Perl.
PHP kalba yra atviro kodo ir tai yra viena priežasčių, dėl ko kalba yra nors ir nesudėtinga, bet gana lanksti – veikia daugumoje operacinių sistemų, palaiko nemažai reliacinių duomenų bazių bei veikia su dauguma interneto serverių – CGI, FastCGI, ISAPI ir kitais protokolais.
